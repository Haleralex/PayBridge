- **Если вы только начали** — читайте по порядку, не перепрыгивая.
- **Если нужно быстро вспомнить термин** — [09-glossary.md](09-glossary.md).
- **Если вы готовите презентацию/доклад** — [11-presentation-outline.md](11-presentation-outline.md).
